Проекты на Python

Конвертер валют

Программа получает актуальные данные о курсе валют и конвертирует в рубли число, заданное пользователем.

Введенное значение обрабатывается и конвертируется из валюты, которую указал пользователь.

Конвертер валют с графическим интерфейсом

В основе лежит HTML-документ, дизайн написан при помощи CSS, а библиотека eel обрабатывает входные данные на Python.
Конвертирование происходит на стороне пользователя при помощи JavaScript.

Работает как приложение.

Пользователь вводит значение. При нажатии на кнопку валюты JavaScript обрабатывает это событие и запускает функцию в Python.
Далее Python переходит по указаннаму url-адресу, получает актуальный курс валюты и возвращает обратно в JavaScript-скрипт.
Полученное значение конвертируется в ту валюту, которую задал пользователь в поле ввода.

Версия с графическим интерфесом

Получает и обрабатывает актуальные данные Python-скрипт.
JavaScript, в свою очередь, позволяет взаимодействовать с полученными данными.


Есть возможность вычислить размер дивидендов (блок слева).
А также вычислить прибыль на акцию, если та (условно) была куплена ранее.
Внизу есть крафик, на котором отображается стоимость акций за последние несколько недель. На текущий момент еще в разработке.
